# Shadcn/ui

This app uses [shadcn/ui](https://ui.shadcn.com/docs/cli) for the component library and style system. This tool provides a CLI which can be used in a workspace like:

```bash
npm exec -w=web -- shadcn all
```

The custom design can be accessed throught this link:
https://ui.shadcn.com/create?style=lyra&iconLibrary=hugeicons&font=jetbrains-mono&theme=amber&radius=none&template=start&baseColor=stone

# Editor

This project will use MDX with custom extensions to emulate much of the features from [Material for Mkdocs](https://squidfunk.github.io/mkdocs-material/reference/).
